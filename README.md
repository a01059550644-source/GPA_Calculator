본 프로젝트는 대학생이 한 학기 동안 수강한 여러 과목의 학점과 평점을 입력받아 평균 학점을 산출하고, 졸업 시 달성하고자 하는 목표 평점을 입력하면 앞으로 남은 학기 동안 최소 몇 학점 이상을 받아야 하는지 수학적으로 계산 및 예측해 주는 파이썬 프로그램입니다.

---

## 주요 기능
1. 여러 과목 성적 동시 수집 : 사용자가 `q`를 입력할 때까지 반복문을 통해 수강한 과목의 이수 학점과 성적을 제한 없이 입력받을 수 있도록 하였습니다.
2. 평균 학점 자동 계산 : 각 과목의 `학점 × 성적` 총합을 총 이수 학점으로 나누는 정석적인 대학교 학점 산출 공식을 파이썬 코드로 직접 구현했습니다.
3. 미래 학점 예측 알고리즘: 유저가 입력한 `졸업 총 학점`과 `목표 학점` 데이터를 기반으로, 남은 이수 학점 대비 앞으로 획득해야 하는 '최소 평균 학점'을 수학적 공식으로 역산하여 출력합니다. 만약 달성 불가능한 점수(4.5 초과)가 계산될 경우 예외 경고 메시지를 띄우도록 설계했습니다.
